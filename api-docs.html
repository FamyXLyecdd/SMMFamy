Introduction

Welcome to the SMMFamy API! You can use our API to programmatically interact with our SMM services, including fetching available services, placing orders, checking order status, and more.

Base URL
https://smmfamy.com/api/v2

Rate Limiting

API requests are limited to 100 requests per minute per API key. Exceeding this limit will result in a 429 error.

Response Format

All API responses are returned in JSON format. Successful responses will contain the requested data, while error responses will include an error field with a description.

Authentication

All API requests require authentication using your API key. Include your API key in every request as a POST parameter.

Getting Your API Key

  1. Log in to your SMMFamy account
  2. Go to Settings or Dashboard
  3. Find your API key in the API section
  4. Keep your API key secret - never share it publicly
// Every request must include: { "key": "YOUR_API_KEY", "action": "services" }

Get Services

Retrieve a list of all available services.

POST /api/v2

Parameters

Parameter Type Description
key required string Your API key
action required string Set to "services"

Response

[ { "service": 1, "name": "Instagram Followers", "type": "default", "category": "Instagram Followers", "rate": "25.00", "min": "100", "max": "100000", "refill": true }, ... ]

Add Order

Place a new order for a service.

POST /api/v2

Parameters

Parameter Type Description
key required string Your API key
action required string Set to "add"
service required integer Service ID
link required string URL of the post or profile
quantity required integer Quantity to order (must be between min and max)
runs optional integer Number of runs for drip-feed
interval optional integer Interval in minutes between runs

Response

{ "order": 123456 }

Order Status

Check the status of one or multiple orders.

POST /api/v2

Parameters

Parameter Type Description
key required string Your API key
action required string Set to "status"
order required integer Order ID

Response

{ "charge": "25.00", "start_count": "1000", "status": "Completed", "remains": "0" }

Request a refill for an eligible order.

POST /api/v2

Parameters

Parameter Type Description
key required string Your API key
action required string Set to "refill"
order required integer Order ID to refill

Response

Cancel Order

Cancel a pending order.

POST /api/v2

Parameters

Parameter Type Description
key required string Your API key
action required string Set to "cancel"
order required integer Order ID to cancel

Response

{ "cancel": 1 } // Or error if order cannot be canceled: { "error": "Order already in progress" }

Error Handling

When an error occurs, the API returns a JSON response with an error field.

Error Message Description
Invalid API key The provided API key is invalid or inactive
Invalid action The requested action is not supported
Not enough balance Your account balance is insufficient
Service not found The specified service ID does not exist
Quantity too low Order quantity is below service minimum
Quantity too high Order quantity exceeds service maximum
Invalid link The provided URL is not valid
Order not found The specified order ID does not exist
Rate limit exceeded Too many requests, please slow down

Code Examples

PHP Example

<?php $api_url = "https://smmfamy.com/api/v2"; $data = [ 'key' => 'YOUR_API_KEY', 'action' => 'add', 'service' => 1, 'link' => 'https://instagram.com/p/xxxxx', 'quantity' => 1000 ]; $ch = curl_init($api_url); curl_setopt($ch, CURLOPT_POST, true); curl_setopt($ch, CURLOPT_POSTFIELDS, $data); curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); $response = curl_exec($ch); $result = json_decode($response, true); print_r($result); ?>

Python Example

import requests api_url = "https://smmfamy.com/api/v2" data = { 'key': 'YOUR_API_KEY', 'action': 'add', 'service': 1, 'link': 'https://instagram.com/p/xxxxx', 'quantity': 1000 } response = requests.post(api_url, data=data) result = response.json() print(result)

JavaScript Example

const apiUrl = 'https://smmfamy.com/api/v2'; const data = { key: 'YOUR_API_KEY', action: 'add', service: 1, link: 'https://instagram.com/p/xxxxx', quantity: 1000 }; fetch(apiUrl, { method: 'POST', headers: { 'Content-Type': 'application/x-www-form-urlencoded' }, body: new URLSearchParams(data) }) .then(res => res.json()) .then(result => console.log(result));
